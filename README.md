# FalconOS

> A 16-bit x86 Real Mode operating system with a colorful boot screen.

```
      ___---___
    //  (   )  \\         F a l c o n O S
   /  (  ~~~  )  \
  |  / O\   /O \  |      v1.0 - COLOR EDITION
  |  |   \ /   |  |
   \ |  _/~~~\_  | /
    \|_/       \_|/        << Born  to  Fly >>
         |   |
        /|   |\
       / |___| \
      ~~~~~~~~~~~
```

## Features

- Pure 16-bit x86 Real Mode bootloader (NASM)
- Colorful boot screen using BIOS INT 0x10 (VGA Text Mode 3, 80×25)
- Dragon logo rendered with multi-color attributes (Bright Red body, Gold fire/eyes)
- OS title in Bright Green, tagline in Cyan
- Clean table-driven string renderer — no libc, no runtime

## Requirements

| Tool | Package (Debian/Ubuntu) |
|---|---|
| `nasm` | `sudo apt install nasm` |
| `qemu-system-i386` | `sudo apt install qemu-system-x86` |
| `git` | `sudo apt install git` |

## Quick Start

```bash
make          # assemble boot.bin
make run      # launch in QEMU
make size     # verify 512 bytes + 0xAA55 signature
make clean    # remove build artifacts
```
